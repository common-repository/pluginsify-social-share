== Description ==
Share post, pages, media, products on social media

== Installation ==
  
  ### Automatic Install From WordPress Dashboard  ###
  1. Login to your the admin panel
  2. Navigate to Plugins â†’ Add New
  3. Search for "Pluginsify Social Share"
  4. Click install and activate respectively. 

== How to use Pluginsify Social Share ==
After successfull installation and activation, the plugin will add a settings page called **Pluginsify Social Share** in dashborad. 
Select and customize require settings. 
To show share bar in custom locaiton inside page use shortcode `[pgfyshare] `
